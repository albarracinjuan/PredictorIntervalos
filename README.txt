Para empaquetar el proyecto con el código fuente suministrado

1. Abrir una terminal de comandos.
2. Ubicarse en la carpeta raíz del proyecto es decir "...\PredictorIntervalos"
3. Ejecutar el comando mvn clean install


Para descargar desde GITHUB y empaquetar proyecto siga estos pasos: 

1. Abrir una terminal de comandos.
2. Ubicarse en el directorio en donde desea descargar el proyecto.
3. Descargar el proyecto utilizando el comando $ git clone https://github.com/albarracinjuan/PredictorIntervalos.git,
   en dónde <nombre_nuevo> es el nombre que le va a dar al proyecto.
4. Por último ubicarse en el directorio del proyecto "directorio seleccionado paso 2/<nombre_nuevo> paso 3/" y ejecutar el comando mvn clean install
